# 5 · Button → CreoPriority

## Описание

Срабатывает по нажатию кнопки в карточке Iteration.  
Создаёт запись в таблице `CreoPriority` с предустановленными полями, связанными с текущей итерацией.

## Триггер

| Поле | Значение |
|---|---|
| Таблица | (таблица с карточками Iteration) |
| Событие | When a button is clicked |

## Действие

- **Create record** в таблице `CreoPriority` с соответствующими предустановками из карточки Iteration

## Скрипт

Скрипта нет — автоматизация построена на встроенном экшене Airtable **Create record**.

> ⚠️ На скриншоте присутствует ошибка тестирования (`Fix testing error`).  
> Проверить корректность маппинга полей при следующем редактировании.

## Связанные таблицы

- `CreoPriority` — создаётся новая запись
